# Mac Mini File Manager — 기능 명세서

## 개요

Tailscale + SSH를 통해 원격 맥미니(kcy-macmini)의 파일을 GUI로 관리하는 데스크탑 앱.
CLI 없이 파일 전송, 탐색, 검색, 삭제 등을 수행한다.

## 기술 스택

- **프레임워크**: Tauri (권장) 또는 Electron
- **프론트엔드**: React + TypeScript
- **SSH 연결**: Rust의 `ssh2` 크레이트 (Tauri) 또는 Node.js `ssh2` 패키지 (Electron)
- **프로토콜**: SFTP over SSH

## SSH 연결 정보

```
Host: kcy-macmini
User: kcy
Port: 22
인증: SSH 키 (권장) 또는 비밀번호
```

## 화면 구조

### 1. 연결 화면 (Connection)

앱 실행 시 최초 화면. 연결 정보를 입력하고 맥미니에 접속한다.

| 요소 | 설명 |
|------|------|
| Host 입력 | 기본값 `kcy-macmini` |
| User 입력 | 기본값 `kcy` |
| Port 입력 | 기본값 `22` |
| 인증 방식 선택 | SSH 키 / 비밀번호 |
| SSH 키 경로 | `~/.ssh/id_ed25519` (파일 선택) |
| 비밀번호 입력 | 비밀번호 선택 시 표시 |
| 연결 버튼 | 클릭 시 SSH 연결 시도 |
| 연결 상태 표시 | 연결 중 / 성공 / 실패 |

- 마지막 연결 정보를 로컬에 저장하여 다음 실행 시 자동 입력
- SSH 키 인증이 설정되어 있으면 앱 실행 시 자동 연결 시도

### 2. 홈 화면 (Home)

연결 성공 후 표시. 주요 기능 버튼이 카드/그리드 형태로 나열된다.

| 버튼 | 기능 | 설명 |
|------|------|------|
| 📤 파일 보내기 | Send | 로컬 파일/폴더를 맥미니로 전송 |
| 📥 파일 가져오기 | Receive | 맥미니 파일/폴더를 로컬로 다운로드 |
| 📁 파일 탐색기 | Browse | 맥미니 파일 시스템을 OS 탐색기처럼 탐색 |
| 🔍 파일 검색 | Search | 맥미니에서 파일명/확장자로 검색 |
| 🗑️ 파일 삭제 | Delete | 맥미니 파일/폴더 삭제 |
| 📊 상태 확인 | Status | 맥미니 디스크 용량, 연결 상태 확인 |

### 3. 파일 탐색기 화면 (File Browser)

맥미니의 파일 시스템을 Finder/탐색기처럼 보여주는 핵심 화면.

#### 레이아웃

```
┌─────────────────────────────────────────────────┐
│  ← →  /Users/kcy/Desktop/docs    [검색] [보기]  │  ← 경로 바 + 네비게이션
├────────────┬────────────────────────────────────┤
│ 즐겨찾기   │  📁 docs          2026-03-15       │  ← 파일/폴더 목록
│  🏠 홈     │  📁 projects      2026-03-14       │
│  🖥 데스크탑│  📄 readme.md     2026-03-13  4KB  │
│  📥 다운로드│  📄 config.json   2026-03-10  1KB  │
│  📄 문서   │                                    │
│            │                                    │
├────────────┴────────────────────────────────────┤
│  4 items | 디스크 여유: 128GB                    │  ← 상태 바
└─────────────────────────────────────────────────┘
```

#### 기능 상세

| 기능 | 동작 |
|------|------|
| 폴더 클릭 | 해당 폴더로 진입 (SFTP `readdir`) |
| 뒤로/앞으로 | 탐색 히스토리 네비게이션 |
| 경로 바 직접 입력 | 특정 경로로 바로 이동 |
| 즐겨찾기 사이드바 | 홈, 데스크탑, 다운로드, 문서 바로가기 |
| 파일 정보 표시 | 파일명, 수정일, 크기 |
| 정렬 | 이름순, 날짜순, 크기순 |
| 보기 모드 | 리스트 / 그리드 전환 |
| 우클릭 컨텍스트 메뉴 | 다운로드, 삭제, 이름 변경, 새 폴더 생성 |
| 드래그 앤 드롭 | 로컬 파일을 앱에 드래그 → 현재 경로에 업로드 |
| 다중 선택 | Cmd+클릭 또는 Shift+클릭으로 여러 파일 선택 |
| 더블 클릭 (파일) | 로컬 임시 폴더에 다운로드 후 기본 앱으로 열기 |

### 4. 파일 보내기 화면 (Send)

```
┌─────────────────────────────────────┐
│  📤 파일 보내기                      │
│                                     │
│  [로컬 파일/폴더 선택]  ← OS 파일 다이얼로그 │
│                                     │
│  선택됨:                             │
│   📄 report.docx (2.3MB)           │
│   📁 images/ (15 files)            │
│                                     │
│  보낼 위치 (맥미니):                  │
│  /Users/kcy/Desktop/docs    [변경]  │  ← 파일 탐색기로 경로 선택
│                                     │
│  [보내기]                            │
│                                     │
│  ████████████░░░░ 67% (1.5MB/2.3MB) │  ← 전송 진행률
└─────────────────────────────────────┘
```

| 기능 | 설명 |
|------|------|
| 파일 선택 | OS 네이티브 파일/폴더 선택 다이얼로그 |
| 다중 파일 선택 | 여러 파일/폴더 동시 선택 가능 |
| 목적지 경로 선택 | 파일 탐색기(화면 3)를 모달로 열어 맥미니 경로 선택 |
| 전송 진행률 | 프로그레스 바 + 퍼센트 + 전송 속도 |
| 전송 완료 알림 | 성공/실패 토스트 알림 |
| 동일 파일명 처리 | 덮어쓰기 / 건너뛰기 / 이름 변경 선택 |

### 5. 파일 가져오기 화면 (Receive)

```
┌──────────────────────────────────────┐
│  📥 파일 가져오기                     │
│                                      │
│  가져올 파일 (맥미니):                │
│  ┌────────────────────────────────┐  │
│  │ 📁 파일 탐색기 (맥미니)         │  │  ← 화면3의 파일 탐색기 임베드
│  │  📁 docs                       │  │
│  │  ✅ 📄 result.txt              │  │  ← 체크박스로 파일 선택
│  │  ✅ 📁 output/                 │  │
│  │  📄 config.json                │  │
│  └────────────────────────────────┘  │
│                                      │
│  저장 위치 (로컬):                    │
│  ~/Downloads              [변경]     │  ← OS 폴더 선택 다이얼로그
│                                      │
│  [가져오기]                           │
│                                      │
│  ████████████████ 100% 완료          │
└──────────────────────────────────────┘
```

| 기능 | 설명 |
|------|------|
| 맥미니 파일 탐색 | 파일 탐색기 UI에서 체크박스로 파일/폴더 선택 |
| 로컬 저장 경로 | OS 네이티브 폴더 선택 다이얼로그 |
| 전송 진행률 | 프로그레스 바 |
| 완료 후 동작 | Finder에서 저장 폴더 열기 옵션 |

### 6. 파일 검색 화면 (Search)

```
┌───────────────────────────────────────┐
│  🔍 파일 검색                         │
│                                       │
│  [검색어 입력________________] [검색]  │
│                                       │
│  검색 위치: /Users/kcy       [변경]   │
│  필터: [모든 파일 ▼] [최근 7일 ▼]     │
│                                       │
│  검색 결과 (3건):                      │
│  📄 ~/docs/report.docx       2.3MB   │
│  📄 ~/Desktop/report-v2.docx 1.8MB   │
│  📄 ~/backup/report.docx     2.1MB   │
│                                       │
│  [선택 항목 다운로드]                  │
└───────────────────────────────────────┘
```

| 기능 | 설명 |
|------|------|
| 검색 실행 | SFTP + `find` 명령어 조합 (`ssh exec`) |
| 검색 범위 지정 | 특정 폴더 하위로 제한 가능 |
| 파일 유형 필터 | 전체, 문서, 이미지, 코드 등 확장자 기반 필터 |
| 날짜 필터 | 최근 1일 / 7일 / 30일 / 전체 |
| 결과에서 바로 동작 | 다운로드, 삭제, 경로 복사, 탐색기에서 열기 |

### 7. 상태 확인 화면 (Status)

| 표시 항목 | 데이터 소스 |
|-----------|------------|
| 연결 상태 | SSH 세션 alive 체크 |
| 맥미니 IP | Tailscale IP (100.x.x.x) |
| 디스크 사용량 | `df -h` 실행 결과 파싱 |
| 디스크 시각화 | 사용량/전체 프로그레스 바 |
| 업타임 | `uptime` 실행 결과 파싱 |
| 재연결 버튼 | 연결 끊겼을 때 재연결 |

## 공통 기능

| 기능 | 설명 |
|------|------|
| 자동 재연결 | SSH 끊어지면 자동 재연결 시도 (3회) |

## SFTP 명령어 매핑

각 GUI 동작이 내부적으로 실행하는 SSH/SFTP 명령어 참고.

| GUI 동작 | 내부 명령어 |
|----------|-----------|
| 폴더 열기 | `sftp.readdir(path)` |
| 파일 업로드 | `sftp.fastPut(local, remote)` |
| 파일 다운로드 | `sftp.fastGet(remote, local)` |
| 폴더 업로드 | 재귀적 `sftp.mkdir` + `sftp.fastPut` |
| 폴더 다운로드 | 재귀적 `sftp.readdir` + `sftp.fastGet` |
| 파일 삭제 | `sftp.unlink(path)` |
| 폴더 삭제 | 재귀적 삭제 후 `sftp.rmdir(path)` |
| 이름 변경 | `sftp.rename(old, new)` |
| 새 폴더 생성 | `sftp.mkdir(path)` |
| 파일 검색 | `ssh.exec('find <path> -name "<query>"')` |
| 디스크 용량 | `ssh.exec('df -h')` |
| 업타임 | `ssh.exec('uptime')` |
